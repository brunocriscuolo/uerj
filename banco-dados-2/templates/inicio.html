Apache CouchDB

O ​CouchDB ​é ​um ​banco ​de ​dados ​não-relacional (NoSQL), orientado ​a ​documentos que armazena objetos tipo JSON. Esse banco de dados utiliza funções MapReduce em linguagem JavaScript, para realização de consultas.
